- Definición de funciones:
	def

- retorno de valores:
	instruccion return

- Argumentos y parámetros

- Argumentos por valor y referencia:
	Paso por valor --- tipo de datos simples (int, float, str, boolean...)
	Paso por referencia --- tipo de datos complejos (diccionarios, conjuntos, tuplas, listas)

- Argumentos y parámetros indeterminados:
	Por posición --- def nombreFuncion(*args) --- tupla
	Por nombre --- def nombreFuncion(**kwargs) --- diccionario

- Funciones recursivas:
	funciones que se llaman así mismas durante su ejecución.

- Funciones integradas:
	round()
	len()
	str()
	int()
	bin()
	hex()
	...
